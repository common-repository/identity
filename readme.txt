== Description ==

**identity** automagically styles all occurences of previously configured words to appear as desired. Now your company name appears uniformly styled throughout your entire website. It's marvellous!

= Main Features =
- Makes enforcing your corporate identity a breeze on your website
- Does all the tedious work of styling certain words on your website
- It sparkles and shines

The current version is available on wordpress.org:
http://downloads.wordpress.org/plugin/identity.zip

Development of the plugin is an open process. Latest code is available on wordpress.org, please report feature requests and bugs on the support forum

== Frequently Asked Questions ==

= How can I participate in improving this plugin? =
- Use it

== Installation ==
- Visit your WordPress Admin area
- Navigate to `Plugins\Add`
- Search for **identity** and find this Plugin among others
- Install the latest version of the **identity** Plugin from WordPress.org
- Activate the Plugin
